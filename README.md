📖 Project Overview
The Cybersecurity Awareness Bot is a comprehensive WPF desktop application designed to educate users about cybersecurity best practices. This is Part 3 of the project, which extends the console-based chatbot from Parts 1 and 2 into a fully-featured GUI application with:

Interactive chat interface with typing animation and colour-coded responses

Task Assistant with JSON data persistence for tracking cybersecurity tasks

Mini Quiz with 8 cybersecurity questions (multiple-choice and true/false)

Activity Log with paginated viewing and file persistence

bar navigation for seamless switching between features

✨ Features
1️⃣ Chat Interface
Keyword Recognition — Detects cybersecurity topics (phishing, passwords, safe browsing, privacy) and provides relevant tips

Random Responses — Multiple pre-defined responses per topic for varied interactions

Sentiment Detection — Identifies user emotions (worried, frustrated, curious) and adjusts responses accordingly

Conversation Memory — Remembers user name and previously discussed topics

Typing Animation — Simulates natural typing for bot responses

Colour-Coded Responses — Visual differentiation for tips, warnings, and success messages

2️⃣ Task Assistant
Add Tasks — Create cybersecurity tasks with optional reminder dates

View Tasks — Display all tasks with status (Pending/Done) and reminders

Complete Tasks — Mark tasks as completed with one click

Delete Tasks — Remove unwanted tasks

JSON Persistence — Tasks are saved to tasks.json and persist across application restarts

3️⃣ Mini Quiz
8 Questions — Mixed multiple-choice and true/false formats

Random Selection — 5 questions drawn randomly per round

Instant Feedback — Correct/incorrect answers with explanations

Score Tracking — Session and total scores maintained

4️⃣ Activity Log
Real-Time Logging — All user and bot actions are logged

Pagination — Shows 5 entries at a time with "Show More" functionality

File Persistence — Logs saved to cyberbot_activity.log in the application directory

Colour-Coded Entries — Visual differentiation for different log types

5️⃣ bar Navigation
💬 Chat — Main conversation interface

📋 Task Assistant — Task management panel

🎮 Mini Quiz — Full quiz interface

📜 Activity Log — Log viewer
